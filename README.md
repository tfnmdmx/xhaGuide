# xhaGuide - 中国海洋大学西海岸校区导航

截至当前（2023/2/3），西海岸校区（还未建设完），现有主流导航软件暂无校区地图，故开发本程序为海大师生提供基础地图服务。

## 功能展示

- 校区平面图贴图
- 校园小公交路线
- 一些常用信息
